### 升级（Upgrade）
1. 下载新版本到部署该项目的根目录
2. 覆盖解压 (tar -zxvf mm-wiki-v0.1.5-mac-amd64.tar.gz)
3. 执行升级命令
```
./mm-wiki --conf conf/mm-wiki.conf --upgrade
```
3. 重新启动
```
./mm-wiki --conf conf/mm-wiki.conf
```
